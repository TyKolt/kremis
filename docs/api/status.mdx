```json 200 OK
  "node_count": 42,
  "edge_count": 35,
  "stable_edges": 10,
  "density_millionths": 1234

```

|-------|------|-------------|
| `node_count` | integer | Total number of nodes in the graph. |
| `edge_count` | integer | Total number of edges. |

| `density_millionths` | integer | Graph density expressed in millionths. |
